The Jf::sql function. The behavior of this function is as follows:
 * On queries with no parameters, it should use query function and fetch all results (no prepared statement) * On queries with parameters, parameters are provided as question marks (?) and then additional function arguments will be 	 bound to question marks. * On SELECT, it will return 2D array of results or NULL if no result. * On DELETE, UPDATE it returns affected rows * On INSERT, if auto-increment is available last insert id, otherwise affected rows
