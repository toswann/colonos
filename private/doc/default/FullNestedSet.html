Class Details

FullNestedSet Class This class provides a means to implement Hierarchical data in flat SQL tables.

FullNestedSet Class This class provides a means to implement Hierarchical data in flat SQL tables. Queries extracted from http://dev.mysql.com/tech-resources/articles/hierarchical-data.html Tested and working properly. Usage: have a table with at least 3 INT fields for ID,Left and Right. Create a new instance of this class and pass the name of table and name of the 3 fields above
